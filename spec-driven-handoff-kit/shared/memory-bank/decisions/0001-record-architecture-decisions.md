# 1. アーキテクチャ決定を記録する

<!-- 最初のADR（兼テンプレート）。新しい決定はこの形式をコピーして 0002, 0003... と連番で追加する。 -->

- ステータス: accepted
- 日付: 2026-06-07
- 置換: なし

## 背景（Context）
AI コーディングエージェントを複数（Copilot / Claude Code / Codex）・複数セッションで使う。
決定を記録しなければ次のセッションや別エージェントで失われ、同じ議論を蒸し返す。

## 決定（Decision）
重要な技術・設計判断を ADR として `memory-bank/decisions/` に1決定1ファイルで連番記録する。
過去のADRは編集・削除せず、変更は **新しいADRで supersede** する（append-only）。

## 結果（Consequences）
- どのエージェントもタスク開始時に過去ADRを読み、確定済み方針を蒸し返さない。
- 「なぜそうしたか」が監査可能になる。
